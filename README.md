#### [Symfony](http://symfony.com)

Symfony2是一个开源的PHP Web框架，有着开发速度快、性能高等特点。与其他框架相比，Symfony2的优势包括：支持DI（依赖注入）和IoC（控制反转）；扩展性强；文档和社区比较成熟。但是Symfony2的学习曲线也比较陡峭，没有经验的初学者往往需要一些练习才能掌握其特性。
